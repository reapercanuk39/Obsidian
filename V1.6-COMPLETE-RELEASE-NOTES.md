## üêõ Bug Fixes

### Critical Issues Resolved
‚úÖ **Boot Path Case Sensitivity** (v1.6-FIXED)
- Fixed "file '/obsidian/vmlinuz' not found" error
- ISO9660 creates UPPERCASE filenames by default
- Updated all GRUB and ISOLINUX configs to use `/OBSIDIAN/` paths
- Tested on physical hardware and virtual machines

‚úÖ **UEFI Boot Failure** (v1.5 ‚Üí v1.6)
- Fixed embedded EFI image with outdated grub.cfg
- Updated both `boot/grub/efi.img` and `efi/efi.img`
- UEFI firmware now successfully boots to GRUB menu

### Minor Improvements
- Initramfs warnings suppressed (cosmetic only)
- VirtualBox keyboard capture issues documented (use physical hardware)
- All broken symlinks to stock kernel fixed

## üìù Installation Notes

### Creating Bootable USB (Linux)
```bash
# Replace /dev/sdX with your USB device
sudo dd if=Obsidian-v1.6-Enhanced-COMPLETE-20260108-0049.iso of=/dev/sdX bs=4M status=progress && sync
```

### Creating Bootable USB (Windows)
Use **Rufus** or **Etcher**:
1. Select ISO file
2. Select USB drive
3. Write in DD mode (recommended) or ISO mode
4. Boot from USB
